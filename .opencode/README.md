# BigPlant OpenCode Workspace

Muc tieu folder nay:
- Chua context he thong de build agent.
- Chua bo skill co the tai su dung cho task hang ngay.
- Chua manifest de map agent <-> skill.

Cau truc:
- `manifest.json`: khai bao agent, skill, file context.
- `context/project_overview.md`: tong quan he thong BigPlant tu source hien tai.
- `agents/*.md`: dinh nghia vai tro agent.
- `skills/*.md`: playbook theo tung nghiep vu.

Nguyen tac update:
- Moi thay doi lon trong source can cap nhat `context/project_overview.md`.
- Neu xuat hien nghiep vu moi, tao them file trong `skills/`.
- Neu doi chien luoc phan tich/trien khai, cap nhat `agents/`.
